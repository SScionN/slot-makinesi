A simple slot game with sound effects


## Features

- 5 Game Rounds
- Maximum Points 30+
- 2 Same Symbols +5
- 3 Same Symbols +10

## Installation

pkg update && pkg upgrade
termux-setup-storage
pkg install python
pkg install mpv
pkg install git
git clone https://github.com/SScionN/slot-makinesi.git
cd slot-makinesi
ls (checks whether the four mp3 files already exist)
python slot.py
